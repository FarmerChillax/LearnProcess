我的阅读清单

计划阅读

  1. 《自己手写一个Java虚拟机》
  2. 《万古江河》
  3. 《CS APP》、《计算机系统》与《算法导论》这三部希望能好好读一下，弥补大学时期的遗憾
  4. 《TCP/IP网络编程》
  5. 《深入理解Linux网络技术内幕》
  6. 《微服务架构设计模式》
  7. 《数据密集型应用系统设计》
  8. 《HTTP权威指南》
  9. 《机器学习》周志华著，又名西瓜书
  10. 《Go并发之道》英文原版
  11. 《Python cook book》现阶段转Go了，以后从事Python相关工作再看
  12. 《High Performance Python》 同《Python CookBook》,暂无精力阅读

2022阅读清单 （进行中…)

  1. 《Kubernetes in Action》5%
  2. 《kubernetes 权威指南》
  3. 《大型网站运维：从系统管理到SRE》
  4. 《Google SRE工作手册》
  5. 《SRE：Google运维解密》
  6. 《分布式缓存 原理、架构及Go语言实现》30%
  7. 《Go语言学习笔记》30%
  8. 《流畅的Python》15% pythoner必读(想要用好Python的人), 给人一种打开新世界的感觉，很Pythonic(豆瓣9.4的高分，无脑推荐无脑入手必不亏)
  9. 《Go 语言核心编程》85% 正如本书前言中所说的，主要讲了类型系统、接口、并发这三部分。(context 与 反射这两部分收获颇丰)
  10. 《七周七并发》30% 很有意思的一本书

2021阅读清单

  1. 《漫画算法：小灰的算法之旅》100%，python版本
  2. 《Python3 网络爬虫开发实战 第二版》50%，在第一版的基础上更新，添加了很多进阶的内容
  3. 《王道408全套》100%，很好的一套书，建议大二、大三阶段看
  4. 《Go 专家编程》100%，内容很丰富很实用，章节顺序安排合理
  5. 《Go web编程》100%，内容过于简单，不太建议
  6. 《Go in action》100%，很多Go语言的栗子
  7. 《剑指offer》40%，拿到offer了就慢慢鸽了
  8. 《Go 并发编程实战》第二版，80%
  9. 《高性能MySQL 第三版》20%，内容很多需要长期阅读
  10. 《Go 语言底层原理剖析》20%，讲解Go语言的底层原理，里面涉及大量的计算机基础，很多干货的一本书
  11. 《Go Web编程实战派》50%，内容很简单适合初学者入门
  12. 《深入浅出Docker》
  13. 《Rust权威指南》官方文档
  14. 《Go语言从基础到中台微服务实战开发》50%，内容不难但写的不好，不推荐
  15. 《Go语言编程之旅》50%，内容很丰富，适合初学者
  16. 《Go语言从入门到进阶实战》
  17. 《乡土中国》
  18. 《从一到无穷大》很符合理工科的口味

2020

  1. 《图解 TCP/IP》30%
  2. 《计算机网络 第七版》谢希仁著，60%
  3. 《鸟哥 Linux 私房菜》30%
  4. 《了不起的Markdown》80%，重点不在语法，而是规范、设计与工具链搭配选择上面
  5. 《Go 语言趣学指南》
  6. 《图解网络硬件》
  7. 《Django Web应用开发实战》
  8. 《数据结构》浙江大学版
  9. 《C语言程序设计》浙江大学版

2019

  1. 《Flask Web 开发实战》200% 贯穿我大学生涯的一本书，web入门到进阶全过程都很有帮助，反复看了5-6遍
  2. 《Python3 网络爬虫开发实战》 100%, 搭配 Flask 与 HTTP 看，更加深刻的理解HTTP协议
  3. 《Python编程：从入门到实践》30%
  4. 《笨办法学python3》30%
  5. 《图解HTTP》100%，很多基本概念需要反复阅读，后面可能会买本《HTTP权威指南》看
  6. 《Python编程从入门到实战》 60% ~ 65%，十分的基础，没有反复看的欲望
