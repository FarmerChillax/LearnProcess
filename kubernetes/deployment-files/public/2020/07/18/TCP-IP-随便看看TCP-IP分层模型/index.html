在TCP/IP的分层中，都会对所发送的数据附加一个首部(包头)，这个首部包含了该层的信息，如目标地址等等。

各层的数据单位：

  • 包(全能性术语)
  • 帧，用来表示数据链路层中包的单位
  • 数据报，网络层以上的分层中包的单位
  • 段，表示TCP流中的信息
  • 消息，指应用协议中的数据单位

数据包的处理

  1. 应用程序处理
    • 将应用数据编码后交给下一层
  2. TCP模块处理
    • 负责建立连接、发送数据以及断开连接。
    • 首部中包括源端口号、目标端口号和序号等
  3. IP模块处理
    • 将TCP的首部和数据段当成IP的数据段，并添加IP首部
    • IP首部中还包含其携带的数据是属于TCP或是UDP
  4. 以太网接口处理

以太网帧格式(Ethernet)

以太网帧前端有一个前导码，它有0、1交替组合而成，表示一个以太网帧的开始
其中前导码末尾是一个叫做SFD的域，值位“11”，前导码和SFD合起来占8个字节。
